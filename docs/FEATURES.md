# Fonctionnalités métier BOD LIGHT

## Génération de prospects

Le site doit convertir les visiteurs en prospects qualifiés.

Fonctionnalités :

- Demande de devis rapide
- Prise de rendez-vous
- Demande de rappel
- Bouton WhatsApp flottant
- Appel direct
- Formulaire de contact intelligent

---

## Présentation des services

Chaque service doit disposer de sa propre page avec :

- Description complète
- Avantages
- Processus de réalisation
- Exemples de réalisations
- Questions fréquentes
- Appel à l'action

---

## Portfolio professionnel

Le portfolio doit permettre :

- Filtrer les réalisations par catégorie
- Voir les images en plein écran
- Afficher les détails de chaque projet
- Présenter le client (si autorisé)
- Présenter les objectifs
- Montrer les résultats obtenus

Catégories :

- Branding
- Graphisme
- Développement Web
- UI/UX
- Marketing Digital
- Réseaux sociaux
- Signalétique
- Impression
- Montage vidéo

---

## Demande de devis

Le formulaire de devis doit permettre :

- Choisir un service
- Décrire le besoin
- Définir le budget
- Définir le délai
- Ajouter des fichiers
- Envoyer automatiquement la demande
